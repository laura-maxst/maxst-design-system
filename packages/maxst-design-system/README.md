# MAXST Design System ui library

## Installation

```
npm i @maxst-designsystem/maxst-design-system
```

example

```react
import { Button } from '@maxst-designsystem/maxst-design-system';

...

<Button isActionMode size="xl" type="primary">
  Button
</Button>

...

```
