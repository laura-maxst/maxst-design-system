## What's Design System?

디지털 제품에서 디자이너와 개발자가 공용으로 활용할 수 있는 하나의 언어 형태로 아래의 항목을 전부 포함하는 ‘시스템 라이브러리’를 의미합니다.

- 디자인 원칙
- UX 패턴
- UI 툴킷
- 코드(Code) 등

## UX 설계 원칙

맥스트 디자인 시스템의 UX는 아래의 원칙을 준수합니다.

<ul className="docs__gray-box-list type-box2">
  <li>
    <h3>통일성</h3>
    <span className="font-body-l">
      맥스트 디자인 시스템은 하나의 시스템을 적용하여 일관된 경험을 제공합니다.
    </span>
    <span className="font-body-l">
      한 사람이 작업한 것처럼 느끼게 해주며, 사용자는 일관된 경험을
      제공받습니다.
    </span>
  </li>
  <li className="font-body-l">
    <h3>효율성</h3>
    <span className="font-body-l">
      맥스트 디자인 시스템은 작업자가 효율적으로 작업할 수 있도록 기여합니다.
    </span>
    <span className="font-body-l">
      시간을 절약하고 능률적으로 작업하며 서로 협업할 수 있습니다.
    </span>
  </li>
  <li className="font-body-l">
    <h3>확장성</h3>
    <span className="font-body-l">
      맥스트 디자인 시스템은 변화하는 흐름에 적응하고, 유연하게 발전합니다.
    </span>
    <span className="font-body-l">
      지속적인 관리로 끊임없이 발전하며 대처합니다.
    </span>
  </li>
  <li className="font-body-l">
    <h3>명확성</h3>
    <span className="font-body-l">
      맥스트 디자인 시스템은 직관적이고 정확한 사용성을 제공합니다.
    </span>
    <span className="font-body-l">
      사용자는 익숙하고 간편한 경험을 제공받을 수 있습니다.
    </span>
  </li>
</ul>

## UI 설계 원칙

맥스트 디자인 시스템의 UI는 아래의 원칙을 준수합니다.

<ul className="docs__gray-box-list">

  </li>

<ol className="docs_ol">

  <li>

    요소들 사이에서 한눈에 찾을 수 있게 해야 합니다.
  </li>
</ol>
