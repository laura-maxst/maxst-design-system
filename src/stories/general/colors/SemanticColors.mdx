# Semantic Color

Base Color를 기반으로 사용 목적에 따른 컬러를 정의합니다.

<br />

## Type

Semantic Color는 컬러 자체의 hex값이 아닌, 사용 목적과 적용하는 인터페이스에 따라 제공합니다.

  <li className="font-body-l">Divider Color</li>
  <li className="font-body-l">Dimmed Color</li>
  <li className="font-body-l">Brand Color(추후)</li>
</ul>

<br />

## Text Color

텍스트 위계에 따라 나누어진 컬러로 사용하며 각 화면 구성 요소에 따라 이외의 컬러를 사용합니다.

## Background Color
