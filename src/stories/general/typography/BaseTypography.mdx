# Base Typography

서비스에 사용하는 공통 타이포그라피를 정의합니다.

<br />
<br />

## Typefaces

Web에서는 Pretendard를 기본으로 하며, APP 환경에서는 기본 OS 서체를 사용합니다.

이에, 웹폰트인 Pretendard는 OS 공통으로 필요한 컴포넌트 등에 한정해 사용합니다.

## Type Scale

Typography를 편의상 ‘font’로 줄여 사용합니다.

사이즈는 2의 배수를 사용하며, 가급적 홀수는 사용을 지양합니다. 또한, 12pt 보다 작은 사이즈는 권장하지 않습니다.

<TypeScaleTable />

## Type Weight

폰트 굵기는 ‘Semibold/Medium/Regular’ 3가지를 사용하고 있습니다. 각각 사용 목적에 따라 적절하게 사용합니다.

<TypeWeightTable />

## Line height

Line height은 기본적으로 배율로 적용되며, 속성에 따라 각각 적용됩니다.

### Title

Title은 가독성 보장을 위하여 1.3배 배율을 사용합니다. (Font size\*1.3)
